# Sales-Performance-Dashboard

📌 Project Overview
This project is a Power BI interactive dashboard built as a hands-on learning exercise to practice data modeling, DAX calculations, and data visualization. The dashboard analyzes sales data by focusing on quantity ordered across timelines, geographical regions, and employee performance.

🛠️ Key Skills & Concepts Practiced
Data Visualization: Built dynamic line-area charts, clustered column charts, multi-row KPI cards, donut/pie charts, and custom image slicers.

DAX Implementation: Practiced writing basic and statistical measures including SUM, AVERAGE, MAX, and STDEV.P (Standard Deviation).

Interactive Filtering: Implemented multi-select month tile slicers and a custom employee profile slicer to enable dynamic data exploration.

📊 Dashboard Breakdowns & Visualizations
1. Total Quantity and Statistical KPIs
Total Quantity by Month (Area-Line Chart): Visualizes the trend of total items ordered over selected months (March, May, August, September, October), highlighting a significant peak in October at 12.5K units.

KPI Cards: Displays global metrics including the absolute total quantity (41K) and a standard deviation calculation (9.86) to evaluate order consistency.

2. Regional & Monthly Distributions
Sum of Quantity Ordered by Month (Column Chart): Provides a direct vertical comparison of monthly sales volume.

Sum of Quantity Ordered by State (Donut Chart): Breaks down product distribution across geographic regions, highlighting CA (California) as the dominant market, followed by MA and NY.

3. Employee Performance & Slicers
Employee Slicer: An advanced visual slicer utilizing custom images and employee names (e.g., Bhushan Bhere, Jignesh Mhatre, Manali Jaygade) to dynamically filter individual metrics.

Performance Matrix: Displays individual key metrics:

Total Quantity (e.g., 41K for the selected view)

First Transactions count (1.176K)

Average Quantity per order (34.96)

Maximum Quantity in a single order (85)

Month Tile Slicer: An intuitive, grid-based layout allowing users to quickly toggle specific months on and off to filter the entire report.
